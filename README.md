这里要做一个运维系统；名字叫:GaussDB运维系统
# 框架：
1.使用django框架搭建
2.暂时使用默认的sqllite
# 系统作用
功能简介:
1.运维单管理系统：用于管理当HCS人员发现gaussdb数据的问题需要定位或者需要咨询时，HCS人员通过该系统创建运维单,经过问题审核、运维人员分析、开发人员分析、开发人员闭环，最后再又运维人员闭环。完成整个运维系统的端到端跟踪。
2.运维人员分配系统：HCS可以通过该系统快速分配到对应的运维人员。
3.开发人员责任田矩阵：运维人员可以根据问题模块通过该矩阵快速找到对应开发人员。
3.运维单分析系统：跟据运维单中的各种信息进行汇总的各种dashbord。
# 其他要求
## UI风格
简洁实用，颜色风格给人严谨专业的感觉。界面布局务实为主。
